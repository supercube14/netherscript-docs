# Getting Started
The basic installation and usage of NetherScript.

## Installation

### Prerequisites

+ [Python 3](https://www.python.org/)
+ Terminal
+ Text Editor

### Required Libraries

The required libaries can be installed with:

```sh
$ pip install -r requirements.txt
```

## Setup

### Windows

There are instructions in `run.cmd` for setting up your Windows registry to be able to double click `.ntscript` files in order to run the compiler.

### Notepad++

You can use `cbscript-npp-highlighting.xml` with [Notepad++](https://notepad-plus-plus.org/) to add syntax highlighting.

### Visual Studio Code

 You can use the [CBScript VS Code extension](https://marketplace.visualstudio.com/items?itemName=spysat.vscode-cbscript) by spysatcorp with [VS Code](https://code.visualstudio.com) to add partial syntax highlighting.
